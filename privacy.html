プライバシーとデータの扱い

サーバーはあなたのデータを保持しません（ゼロリテンション）

このサービスのサーバーは、あなたが入力した会議・案件・報連相の本文を保存しません。データベースもファイル書き込みもありません。AIアシストのために一時的に処理するだけで、応答を返し終えたら破棄します。

「ゼロリテンション」とは、サーバー側にユーザーデータを一切ためない設計方針です。あなたの作業データは、あなたのブラウザの中だけに残ります。

データはどこに保存されるのか

案件（A3）・会議・議題・決定・申し送り・アクションといったあなたの作業データは、すべて あなたのブラウザの localStorage に保存されます。サーバーには送られません。

  • 別の端末やブラウザからは見えません（同期しません）。
  • ブラウザの保存データを消去すると、作業データも消えます。
  • 共有端末では、使い終わりにブラウザデータを消去することをおすすめします。

AIアシストのときだけ、対象テキストが送信されます

AIアシスト（報連相の自動仕分け、議題の添削、要約、なぜなぜ深掘りなど）は、あなたが明示的にボタンを押した操作のときだけ動きます。自動では送信しません。

送信されるもの

  • そのアシストの対象となるテキスト（例: 仕分けしたい文章、添削したい議題）のみ。
  • アシストに必要な最小限の付随情報（context）。

送信されないもの

  • localStorage に貯まった過去の案件・会議・アクションなどのデータ一式。
  • 氏名・メールアドレスなどのアカウント情報（そもそも取得しません。ログインもありません）。

送信された対象テキストは、AIの応答を生成するためだけに使われ、サーバー側には保存されません。

ログはメタデータのみ

サーバーの運用ログには、リクエストのメタデータだけを記録します。入力本文やAIの応答本文は決してログに出力しません。

  • 記録するもの: 日時、呼び出したエンドポイント名、HTTPステータス、入力の文字数、処理時間。
  • 記録しないもの: 入力本文、AIの応答本文、個人を特定する情報。

リバースプロキシ（前段のWebサーバー）を挟んで運用する場合は、そのアクセスログにリクエストボディを残さない設定にしてください。詳細は docs/DEPLOY.md を参照してください。

外部AIサービスについて

AIアシストは Anthropic の Claude API を利用します。実運用では、送信された対象テキストが Anthropic のAPIへ転送され、応答生成に使われます。Anthropic 側の取り扱いは同社のポリシーに従います。モック動作（MOCK_LLM）では外部送信は一切行いません。
